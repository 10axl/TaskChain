415     *
416     * It's important you don't perform blocking operations in this method. Only use this if
417     * the task will be scheduling a different sync operation outside of the TaskChains scope.
418     *
419     * Usually you could achieve the same design with a blocking API by switching to an async task
420     * for the next task and running it there.
